# Additional Field Info

## Description

This mod displays additional information in the field info panel when walking through a field in Farming Simulator 25:

- Area of each field in the current land
- Current land area
- Price per hectare (or acre) of the cultivated area on land you don't own
- Potential harvest quantity and yield on ready to harvest fields
- Crop growth stage

## Features

- Multiplayer support
- Translations available: English, French, German, Polish, Russian, Portuguese, Czech, Ukrainian, Spanish, Brazilian Portuguese
- Improved accuracy for potential harvest quantity and yield
- Shows crop growth stage

## Installation

1. Download the mod file and place it in your Farming Simulator 25 `mods` folder.
2. Enable the mod from the in-game mods menu.

## Usage

When walking through a field, the info panel will show extra data about the land and crops, making it easier to manage and plan your farming activities.
